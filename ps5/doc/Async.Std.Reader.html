Module Async.Std.Reader

module Reader: sig .. end
The Reader module (not to be confused with Pipe.Reader) is for doing network and file input.
type t
A Reader.t is a handle to a file or network stream.
val file_lines : string -> string list Deferred.t
file_lines filename returns a list of the lines in the named file. The lines do not contain the trailing newline.
val file_contents : string -> string Deferred.t
file_contents filename returns the string with the full contents of the file
val read_line : t -> [ `Eof | `Ok of string ] Deferred.t
reads a single line from the given input stream. Returns `Eof if the stream was closed before the line could be read.
